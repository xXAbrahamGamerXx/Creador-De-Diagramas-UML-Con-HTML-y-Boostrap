# Creador-De-Diagramas-UML-Con-HTML-y-Boostrap
Plataforma web MVC para crear, editar y gestionar diagramas UML con SVG. Incluye autenticación, proyectos colaborativos, interfaz responsiva (Bootstrap) y almacenamiento estructurado. Escalable para futuras funcionalidades. Orientada a educación e ingeniería de software.
