# Creador de Diagramas UML

El presente proyecto consiste en una plataforma web para la creación, edición y gestión de diagramas UML, desarrollada bajo el patrón de arquitectura Modelo-Vista-Controlador (MVC), lo que permite una estructura organizada, mantenible y escalable del sistema.

La aplicación fue desarrollada utilizando tecnologías web como HTML, CSS, Bootstrap y JavaScript, proporcionando una interfaz moderna, intuitiva y adaptable a distintos dispositivos. Para la representación gráfica de los diagramas se empleó tecnología SVG (Scalable Vector Graphics), permitiendo que cada elemento generado sea completamente editable mediante código, facilitando modificaciones dinámicas, personalización y almacenamiento eficiente de los diagramas.

El sistema está orientado al trabajo colaborativo mediante proyectos, donde cada usuario cuenta con un sistema de autenticación y gestión de cuentas propio. Los diagramas se organizan dentro de proyectos, permitiendo una administración más eficiente de los recursos y favoreciendo el desarrollo estructurado de modelos de software.

La plataforma ha sido diseñada con un enfoque escalable, permitiendo la incorporación futura de nuevas funcionalidades, tipos de diagramas UML, herramientas colaborativas, control de versiones, permisos avanzados de usuarios e integración con servicios externos mediante APIs.

Entre sus principales características destacan:

* Registro e inicio de sesión de usuarios.
* Gestión de proyectos y diagramas UML.
* Creación y edición visual de diagramas mediante SVG.
* Arquitectura MVC para una mejor organización del código.
* Interfaz responsiva desarrollada con Bootstrap.
* Almacenamiento estructurado de información.
* Sistema preparado para futuras ampliaciones y nuevas funcionalidades.
* Diseño orientado a entornos educativos y profesionales de ingeniería de software.

Este proyecto busca proporcionar una herramienta accesible y eficiente para el modelado UML, facilitando el análisis, diseño y documentación de sistemas de software mediante una plataforma web moderna y extensible.
